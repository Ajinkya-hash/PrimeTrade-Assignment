# Primetrade.ai Assignment

## Business Question
How does market sentiment influence trader performance and behavior?

## Analysis Included
- EDA
- Sentiment vs PnL
- Win Rate Analysis
- Trade Size Analysis
- Trader Segmentation
- KMeans Clustering
- Statistical Testing
- Actionable Trading Rules

## Key Insights
1. Extreme Greed delivers the highest profitability.
2. Trader behavior changes with sentiment.
3. Active traders generally outperform less active traders.
4. Sentiment can improve risk management.

## Tech Stack
Python, Pandas, Matplotlib, Scikit-Learn, SciPy
